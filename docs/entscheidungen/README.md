# Entscheidungen (ADR – Architecture Decision Records)

Projektentscheidungen werden als **Architecture Decision Records (ADR)** dokumentiert. Jede Entscheidung erhält eine eigene nummerierte Datei.

## Status-Werte
| Status | Bedeutung |
|---|---|
| `In Diskussion` | Entscheidung wird noch diskutiert |
| `Entschieden` | Entscheidung ist getroffen und gültig |
| `Verworfen` | Entscheidung wurde revidiert (Nachfolge-ADR verlinken) |

## Namenskonvention
`ADR-[NNN]_[Kurztitel].md` (NNN = dreistellige Nummer, z.B. 001, 002, ...)
